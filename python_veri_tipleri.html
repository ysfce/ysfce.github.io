Python Veri Tipleri

Veri Tipleri

Değişkenlerde depoladığımız verilerin çeşitleri vardır. Örneğin merhaba gibi bir değer string veri tipiyle depolanırken 15 gibi bir değer integer veri tipiyle depolanır.

Aşağıda bazı veri tipi örnekleri verilmiştir:

 string : Metin Veri Tipi
 integer : Tam Sayı Veri Tipi
 float : Kesirli Sayı Veri Tipi
 boolean : Doğru-Yanlış Veri Tipi

Veri Tipi Oluşturma

Herhangi bir değişken tanımladığımızda biz belirtmesek bile python o veri tipinin türünü algılar. Bunu görmek için type fonksiyonunu kullanabiliriz.

type fonksiyonu içine verdiğimiz bir değişkenin veri tipi hakkında bilgi verir. Bunu kullanmak için şöyle bir komutu çalıştırabiliriz:

x = 15
type(x)

Örnekte ilk önce bir x değişkeni oluşturduk. Daha sonra type fonksiyonu ile değişkenin içindeki değeri öğrenmeye çalıştık. Fakat böyle bir kullanımda herhangi bir output alamadık. Önceki derslerde öğrendiğimiz print fonksiyonunun aslında çok basit bir işlevi olsada programlarımızda diğer fonksiyonlarla birlikte kullanmamız gerekebiliyor.

type fonksiyonunun sonucunu görmek içinde kodumuzu şu şekilde değiştirmemiz gerekmekte:

x = 15
print(type(x))

<class 'int'>

Bu değişiklik sonucunda print fonksiyonunu da kullanarak değişkenimizin veri tipini öğrenebildik.

Output kısmındaki class değişkenin tipi hakkında bilgi verildiğini belirtirken int ise tam sayı anlamına gelen integer'ın kısaltmasıdır.
